## Описание

Проект реализует прогнозирование временных рядов продаж с использованием синтетических данных. Используются модели XGBoost и ARIMA/SARIMA для анализа трендов, сезонности и остатков.

## Требования

- Python 3.8+

- Библиотеки: `numpy`, `pandas`, `matplotlib`, `scikit-learn`, `xgboost`, `statsmodels`, `pmdarima`
